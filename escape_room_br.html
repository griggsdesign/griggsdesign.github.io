13th Gate Escape

Placing Vases

I constructed and programmed a network of ten RFID reader chips for less than $100 to check the order of ten colored vases against a solution key, all fitting within a slim 1/2" form factor. In trying to network all ten cheaply-made RFID readers to a single MCU (microcontroller), I discovered that their SS (slave select) pins were not functional. I pivoted, wiring a seperate MCU to each RFID reader, then networking them to a single master detecting ten correct/incorrect parallel status bits.

I later devised an alternative solution with custom code using multiplexers for each data line to allow up to sixteen RFID readers to report to a single MCU, which greatly reduced project complexity and cost. I shared my solution and hacked code on GitHub, and as a result a new friend in Berlin owes me dinner, and I have recently started collaborating on an impressive device called The Read Read.

Stepping Stones

I wired and programmed a trick floor with thirty-six individual pressure-sensing stepping stones. Other workers crafted the stones themselves. I designed the spring suspension and pressure switch system, tuning each stone to preserve realism (stones with springs that are too long oscillate and feel fake) while maintaining reliability (springs that are too short over time allow the stone to rest on the switch, causing false triggers). I wrote code to require an exact sequence of solution stones. With each wrong step, the puzzle resets, and the ceiling lowers, inching closer to "crushing" anyone inside (of course it stops at a safe distance).
